# 1. Seletores:

### [Element](http://tonylampada.github.io/eunaoaguentomaisnaosabercss/seletores/type.html)

```css
/* Isso seleciona a porra toda */
* {
	...
}

/* Todos os <h1>'s */
h1 {
	...
}

/* Todos os <p>'s que forem descendentes de <span>'s */
p span {
	...
}

```
